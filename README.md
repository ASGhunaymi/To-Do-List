# To-Do-List
Simple Python CLI To Do List application with SQLite

---

## Features:
- View Tasks
- Add Tasks
- Remove Tasks
- Tasks are saved in SQLite Database so you won't lose them
- Easy-to-use CLI application 

---

## How To Run:
1. Make sure that Python is installed on your system  
2. Copy or download this repository  
3. Run the program:  

```bash
python to_do_list.py
```
Follow the on-screen menu to manage your tasks
